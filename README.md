## Usage

```
git clone https://github.com/jordan-kasper/vue-music-app.git (vue-music-app)
cd vue-music-app
npm install 
npm run serve 
```

Go to http://localhost:8080/

## Test App

### Run test suit along with lint
```
npm run test
```

### Run unit tests
```
npm run test:unit
```

### Run your end-to-end tests
```
npm run test:e2e
```

### Lints and fixes files
```
npm run lint
```
